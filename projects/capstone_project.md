# Capstone Project

Buat dashboard penjualan berbasis `toko_pintar` dengan minimal 10 query. Laporan akhir harus memuat pertanyaan bisnis, query, output, interpretasi, risiko kesalahan, dan rekomendasi keputusan.

Topik wajib: omzet, jumlah pesanan, produk terlaris, pelanggan bernilai tinggi, kota pelanggan, status pembayaran, status pengiriman, stok rendah, rekonsiliasi, dan performa query.
