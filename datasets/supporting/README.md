# Dataset Pendukung v4

Folder ini berisi data sintetis untuk latihan lanjutan. File CSV tidak wajib dimuat oleh aplikasi browser, tetapi disiapkan untuk praktik DBMS nyata seperti SQLite, MySQL, PostgreSQL, dan DuckDB.

Isi utama:
- `produk_catalog_2000.csv`: katalog produk sintetis.
- `pelanggan_synthetic_3000.csv`: data pelanggan sintetis.
- `daily_sales_2026.csv`: data penjualan harian untuk latihan dashboard dan forecasting sederhana.
- `warehouse_stock_snapshot.csv`: snapshot stok gudang untuk latihan inventory control.
