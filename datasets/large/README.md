# Dataset Besar Opsional

CSV pada folder ini dibuat untuk latihan skala besar: import data, indexing, agregasi, window function, query plan, dan optimasi. File tidak dimuat oleh aplikasi utama agar GitHub Pages tetap cepat.

Kolom: order_id, customer_id, product_id, order_date, status, quantity, unit_price, discount, city, payment_method, device, session_key, batch_hash.

Gunakan dataset ini pada DBMS lokal, misalnya PostgreSQL/MySQL/SQLite, bukan pada engine browser.
