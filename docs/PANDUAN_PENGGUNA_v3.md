# Panduan Pengguna v3.0

Buka `index.html`, isi nama peserta, lalu mulai dari modul pertama atau pilih modul melalui sidebar. Gunakan playground SQL untuk menjalankan contoh query. Setiap modul memiliki materi, studi kasus, latihan, quiz, dan checklist.

Progress tersimpan otomatis di storage internal browser. Apabila berpindah perangkat, gunakan menu Tools untuk export progress JSON lalu import pada perangkat lain.

Sertifikat aktif setelah nama peserta terisi, seluruh modul selesai, seluruh latihan selesai, dan seluruh quiz dijawab.
