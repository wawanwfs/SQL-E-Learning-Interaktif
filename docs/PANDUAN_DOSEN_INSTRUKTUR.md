# Panduan Dosen dan Instruktur

Paket ini dapat digunakan sebagai modul praktikum SQL. Instruktur dapat mengatur pembelajaran menjadi 14 pertemuan, bootcamp intensif, atau belajar mandiri. Setiap pertemuan sebaiknya memuat pembukaan konsep, demonstrasi query, praktik mandiri, review output, dan refleksi.

Gunakan folder `sql/` untuk script praktikum, folder `data/` untuk challenge bank, dan folder `projects/` untuk tugas akhir. Penilaian disarankan menggunakan rubrik akurasi query, validasi output, interpretasi, keterbacaan, dan dokumentasi asumsi.
