# Panduan Pengembangan Konten

Konten utama berada pada `assets/js/data.js` dan versi JSON berada pada `data/lesson_bank.json`. Setiap modul memiliki elemen berikut: tujuan pembelajaran, teori, konsep inti, query utama, latihan, quiz, checklist, kesalahan umum, catatan profesional, dan tugas proyek mini.

Untuk menambah modul baru, tambahkan objek lesson dengan `id` unik. Setelah itu, tambahkan query latihan pada folder `sql/` bila ingin menyediakan versi SQL terpisah.
