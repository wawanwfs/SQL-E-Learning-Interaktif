# Panduan Instalasi

Proyek ini tidak membutuhkan backend. Untuk membuka secara lokal, jalankan salah satu cara berikut.

```bash
python -m http.server 8000
```

atau buka langsung file `index.html` di browser.

Untuk pengujian DBMS penuh, jalankan `sql/database_toko_pintar.sql` pada SQLite, MySQL, atau PostgreSQL dengan penyesuaian kecil pada tipe data jika diperlukan.
