# Changelog v3.0

Versi ini memperluas sistem dari v2.1 menjadi paket GitHub Pages yang lebih lengkap. Jumlah modul diperluas sampai 100, fitur penyimpanan lokal ditambah, pusat tools disediakan, dan data pendukung diperbanyak. Fokus perubahan mencakup UI/UX, progress, sertifikat, catatan, bookmark, export/import, drill acak, badge, serta paket dokumen pendukung.
