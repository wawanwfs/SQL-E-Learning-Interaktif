# Panduan Pengguna v4

1. Buka `index.html` melalui GitHub Pages.
2. Isi nama peserta pada awal penggunaan. Nama, progress, catatan, jawaban, dan pengaturan disimpan di localStorage browser.
3. Gunakan sidebar untuk memilih modul.
4. Jalankan query pada playground.
5. Selesaikan materi, latihan, dan quiz.
6. Gunakan menu v4 untuk planner, ujian, flashcards, dataset, dan export transkrip.
7. Sertifikat dapat dicetak setelah syarat kelulusan terpenuhi.

Catatan: aplikasi ini statis. Semua data personal pengguna tersimpan lokal di perangkat pengguna, bukan di server.
