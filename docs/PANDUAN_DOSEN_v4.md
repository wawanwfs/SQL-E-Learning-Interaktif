# Panduan Dosen/Instruktur v4

Paket ini dapat dipakai untuk pembelajaran mandiri, praktikum kelas, atau pelatihan. Rekomendasi penggunaan:

- Pertemuan 1–4: fondasi SQL dan CRUD.
- Pertemuan 5–8: JOIN, agregasi, subquery, constraint.
- Pertemuan 9–12: index, view, trigger, transaksi, optimasi.
- Pertemuan 13–16: CTE, window function, warehouse, security, capstone.

Instruktur dapat menggunakan `data/v4_exam_bank.json`, `data/v4_casebook.json`, `sql/assessment_queries_v4.sql`, dan `docs/USULAN_PENGEMBANGAN_100_v4.md` sebagai bahan asesmen.
