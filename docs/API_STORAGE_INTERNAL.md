# Storage Internal Browser

Aplikasi menggunakan `localStorage` dengan prefix `sql-master-v30:`. Data yang disimpan meliputi nama peserta, tema, modul terakhir, progress sesi, progress latihan, jawaban quiz, draft SQL, catatan, bookmark, riwayat query, preferensi tampilan, dan ID sertifikat.

Tidak ada data yang dikirim ke server karena aplikasi berjalan statis di GitHub Pages.
