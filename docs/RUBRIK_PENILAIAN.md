# Rubrik Penilaian SQL

| Aspek | Bobot | Indikator |
|---|---:|---|
| Akurasi Query | 30% | Query berjalan dan menghasilkan output sesuai kebutuhan. |
| Validasi Output | 20% | Mahasiswa memeriksa jumlah baris, NULL, agregasi, dan duplikasi. |
| Interpretasi Bisnis | 20% | Output diterjemahkan menjadi informasi dan rekomendasi. |
| Keterbacaan SQL | 15% | Alias, indentasi, dan struktur query mudah dibaca. |
| Dokumentasi Asumsi | 15% | Batasan data dan asumsi query dijelaskan. |
